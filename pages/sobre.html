Sobre o EcoQuest

O EcoQuest é o projeto desenvolvido pela equipe para o Desafio 2 da SoulUp no FIAP Challenge 2026.

O problema que o projeto resolve

Muitas pessoas têm dificuldade de pagar o transporte público no dia a dia. O custo das passagens pesa bastante no orçamento de quem tem renda baixa.

Além disso, a plataforma SoulUp teve uma queda de 77% nos usuários ativos em 2026. As pessoas precisam de um incentivo real para continuar usando.

O EcoQuest tenta resolver esses dois problemas ao mesmo tempo: incentiva ações sustentáveis e dá créditos de transporte como recompensa.

O que o EcoQuest faz

Pontos com valor real

As ações sustentáveis geram pontos que podem ser convertidos em créditos de transporte público. 100 pontos = R$ 1,00.

Voucher de transporte

No início o sistema gera vouchers digitais. A ideia é evoluir para integração direta com o Cartão BOM e o Bilhete Único.

Validação das ações

O sistema verifica as ações antes de creditar os pontos para evitar que usuários ganhem pontos sem fazer nada.

Cálculo de CO2 evitado

Para cada viagem financiada pelos pontos, o sistema calcula quanto CO2 foi evitado e mostra isso para o usuário.
