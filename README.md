Sobre o EcoQuest

O EcoQuest é o projeto desenvolvido pela equipe para o Desafio 2 da SoulUp no FIAP Challenge 2026.

- O problema que o projeto resolve:

Muitas pessoas têm dificuldade de pagar o transporte público no dia a dia. O custo das passagens pesa bastante no orçamento de quem tem renda baixa.

Além disso, a plataforma SoulUp teve uma queda de 77% nos usuários ativos em 2026. As pessoas precisam de um incentivo real para continuar usando.

O EcoQuest tenta resolver esses dois problemas ao mesmo tempo: incentiva ações sustentáveis e dá créditos de transporte como recompensa.
